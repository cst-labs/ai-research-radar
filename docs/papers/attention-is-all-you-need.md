# Attention Is All You Need

**Link:** [https://arxiv.org/abs/1706.03762](https://arxiv.org/abs/1706.03762)

**Authors:** Ashish Vaswani, Noam Shazeer, Niki Parmar, Jakob Uszkoreit, Llion Jones, Aidan N. Gomez, Lukasz Kaiser, Illia Polosukhin

**Date:** 2017-06-12

**Type:** `paper`

**Status:** `reviewed`

**Topics:** [LLM Reasoning](../topics/llm-reasoning.md), [Model Architecture](../topics/model-architecture.md)

## Takeaway

Self-attention became the foundation for modern large language models.

## Summary

Introduced the Transformer architecture, replacing recurrent sequence modeling with multi-head self-attention and feed-forward blocks.
