# Compositional Skill Routing for LLM Agents: Decompose, Retrieve, and Compose

**Date:** 2026-06-22

**Type:** `paper`

**Status:** `unread`

**Primary Topic:** [Agent Systems](../topics/agent-systems.md)

**Topics:** [Agents](../topics/agents.md), [Agent Systems](../topics/agent-systems.md), [LLM Reasoning](../topics/llm-reasoning.md)

## Takeaway

Skill routing treats agent capability as a compositional systems problem rather than a single prompt or model call.

## Summary

Proposes a routing approach where LLM agents decompose tasks, retrieve relevant skills, and compose them to solve complex requests.
