**Authors:** Theodora Worledge, Albert Jiang, Karthik Narasimhan, Ofir Press

**Date:** 2026-05-28

**Type:** `paper`

**Status:** `unread`

**Topics:** [Agents](../topics/agents.md), [Agent Systems](../topics/agent-systems.md), [AI Infra](../topics/ai-infra.md)

## Takeaway

The system wrapped around the model is becoming a primary scaling axis for agent performance.

## Summary

Argues that agentic AI progress increasingly depends on scaling the harness around models, including tools, environment design, and execution scaffolds.
